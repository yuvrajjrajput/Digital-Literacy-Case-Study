## Reflection 

For Task 2, I set up professional profiles on three platforms — GitHub, LinkedIn,
and Kaggle. Each platform serves a distinct purpose in building a student's digital
presence.

GitHub is primarily used by developers and students to host code, collaborate on
projects, and showcase their technical work. Over the next four years, I plan to use
GitHub to store all my programming assignments, personal projects, and eventually
contribute to open-source repositories.

LinkedIn is a professional networking platform used by employers, recruiters, and
students worldwide. I filled in my Education section with my current degree and
institution. Over time, I plan to add internship experiences, certifications, and
project links so that potential employers can easily find my profile.

Kaggle is a platform focused on data science and machine learning, offering datasets,
competitions, and learning courses. As I progress through my Mtech.AI, I plan to
participate in beginner competitions and use Kaggle's free courses to build skills
in Python and data analysis.

Setting up these profiles early means I can build them up gradually over four years
rather than scrambling to create them at placement time.
